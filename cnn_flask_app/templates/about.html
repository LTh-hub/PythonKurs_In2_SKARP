About, vad är detta!

Det här programmet låter dig ladda upp ett foto, därefter uppskattas innehållet som om det föreställer antingen hund eller katt. Resultatet presenteras som Hund eller Katt, samt visar med vilken sannorlikhet modellen baserar dess uppskattning.

Modellen som sannorlikhetsuppskattning kommer från är en CNN modell, Convolutional Neural Network, som är speciellt lämpliga för sådan här typ av klassificering. Den är uppbyggd av fyra Conv2D lager där antalet filter, noder eller neuroner, varierat som 32, 64, 256 resp sista djupaste lager på 256 noder. Användes 15 epochs för anpassning och konvergens. Modelleringen tränades fram på 7700 bilder med katter och hundar, ytterligare 880 bilder användes till modelltest. Modellen har uppvisat Accuracy på datat (träna/test) som (0.98, 0.85). I förluster som värdet Loss blev motsvarande (0.066, 0.453).

Bildmaterial som använts för att lära modellen hur uppdelning ska utföras mellan hundar och katter har till stor del laddats ned från nätet, och i synnerhet från kaggle, som tillhandahåller mycket matnyttigt för Machine Learning.

För datahantering, anpassning av kod och NN-utvecklingen har allt modellbaserat förarbete utförts mha jupyter notebook. Open source och kraftfulla funktioner från TensorFlow som gjort att modellering med neurala nätverk blivit ganska enkelt genomförbart, men som sagt, skit in - skit ut. Versionsmässigt så blev det ganska knepigt att porta modellen från jupyter till VS Code. Varken nya eller gamla versioner var kompatibla. Med Python kernel ver. 3.10.11 och TensorFlow ver. 2.19.0 lyckades konststycket med rull på helheten mot webgränssnittet.

Det web baserade gränssnittet har tagits fram via Flask under Python i VS Code miljö. Meny systemet har länkats in som Bootstrap.

Programmet här är en avslutande inlämningsuppgift i en programmeringskurs påYrkersHögskolan, Göteborg: Pythonprogrammering för AI-utveckling / Egna projektet - del 2, åtta veckors distansutbildning - april-maj 2025.
